# Fixr

A comprehensive home service platform connecting clients with local service providers across the Philippines. Fixr streamlines the process of finding, hiring, and receiving professional home services while empowering the informal sector by providing reliable opportunities for local workers.

## üìã Project Summary

Fixr is a full-stack web application that bridges the gap between clients seeking home services and skilled service providers. The platform operates on a dual-user system where clients can easily locate and book professionals for various services (cleaning, repairs, wellness, etc.), while workers can register to offer their services and connect with potential clients. Built with a modern React frontend and PHP backend, Fixr features real-time location tracking, booking management, chat functionality, and an administrative dashboard for platform oversight.

## ‚ú® Key Features

### For Clients
- **Service Discovery**: Browse and search for various home service categories
- **Real-time Location Tracking**: Find nearest available service providers using Google Maps integration
- **Service Booking**: Easy-to-use booking system with status tracking (Pending, In-Progress, Completed)
- **Direct Communication**: Built-in chat system to communicate with service providers
- **Feedback System**: Submit feedback and rate services
- **Booking Management**: Track service history and current bookings

### For Workers
- **Professional Registration**: Create profiles with service categories and expertise
- **Availability Status**: Set status (Available, Busy, Do Not Disturb, etc.)
- **Booking Management**: View and manage incoming service requests
- **Location Sharing**: Share real-time location with clients during service
- **Profile Management**: Update professional information and service offerings

### For Administrators
- **Dashboard Analytics**: Comprehensive overview of platform metrics
- **User Management**: Manage both client and worker accounts
- **Service Management**: Add, edit, and monitor service categories
- **Report Handling**: Review and take action on user reports
- **User Moderation**: Suspend or warn users when necessary

### General Features
- **Dual Role System**: Users can register as either clients or workers
- **Authentication & Authorization**: Secure JWT-based authentication
- **Responsive Design**: Mobile-friendly interface
- **Real-time Updates**: Live booking and chat notifications
- **Location Services**: Geolocation-based service provider matching
